# GeoBook — Web-версия книжного магазина

Веб-приложение книжного магазина, разработанное на базе фреймворка Flask (Python) в рамках производственной практики.
Проект реализует два интерфейса (пользовательский и административный), систему базы данных книг, 
сортировку, покупку и аренду книг, а также автоматические напоминания об окончании срока аренды.

## Используемые технологии

### Backend
- Python 3
- Flask
- Flask-Login
- Flask-SQLAlchemy
- SQLite
- APScheduler (для автоматических напоминаний)

### Frontend
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- tsParticles (анимированный фон)

Приложение построено по архитектуре Flask:
- маршруты (routes)
- модели данных (ORM SQLAlchemy)
- шаблоны (Jinja2)
- статические файлы


## Функциональные возможности

### Пользовательский интерфейс

- Регистрация и авторизация
- Просмотр каталога книг
- Сортировка по:
  - категории
  - автору
  - году написания
- Просмотр детальной страницы книги
- Покупка книги
- Аренда книги:
  - 2 недели
  - 1 месяц
  - 3 месяца
- Просмотр своих заказов
- Получение уведомлений о завершении аренды

---

### Административный интерфейс

- Просмотр статистики (книги, пользователи, аренды)
- Добавление новых книг
- Редактирование:
  - названия
  - автора
  - категории
  - года
  - описания
  - цен (покупка и аренда)
  - статуса
  - доступности
- Удаление книг
- Просмотр всех аренд
- Отправка напоминаний вручную
- Автоматические напоминания об окончании аренды

## Установка и запуск

Для запуска необходимо установить Python 3 (рекомендуется 3.9+), затем выполнить следующие команды:

```bash
git clone https://github.com/ВАШ_НИК/geobook-store.git
cd geobook-store
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python seed.py
PORT=5001 python app.py

После запуска приложение доступно по адресу:

http://127.0.0.1:5001

Тестовый администратор:
логин: admin
пароль: admin
